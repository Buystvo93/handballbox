About HandballBox

HandballBox is a developing global database for handball players, clubs, leagues, matches, and transfers. The goal is to create a complete, structured, and searchable system similar to Volleybox, but focused entirely on handball.

This project is currently in early development. Data structure, pages, and features are being built step by step.
